# signUpPage

This project is part of odin project web development curriculum [odin project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-sign-up-form).

This project involves creating a sign up form, utilizing buttons and other form elements to take users input, emails and password to create a demo account

## what skills does this project test?

This project tests the following skills:

- Form creation and validation
- CSS styling and layout
- User input handling
